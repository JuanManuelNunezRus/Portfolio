About Me

I’m a Game designer and programmer who wants to continue learning and working in the Gaming Industry. Videogames have been my number one passion during my entire life, and working on them as my job either designing them or coding them is my go-to goal

Projects

DESOLATIUM

First commercial game I ever worked on. Point and click first person adventure where the player has to pick up items, talk to NPCs and solve puzzles.
Joined mid development and worked on programming game mechanics, UI (navigation, behaviour, etc) and refactoring the code to make it easier to finish the development.
Also participated on patching the game for consoles.

Magical Bakery

Store managing game in which I joined in the latter stages of the game.
Helped mainly with bugfixing the multiple versions of the game, both for PC and consoles versions.

Petit Island

Chill game where the player has to complete different tasks to help the island's population and discover all the secrets of Petit Island.
Main programmer for all consoles versions of the game (PS5, Xbox Series & Nintendo Switch), including tasks such as implementing console specific features and improving the game's performance.

Spooky Hat Shop

Game developed for the Master's thesis. Spooky Hat Shop is casual game in which you have to manage a hat shop for ghosts. Gameplay-wise it's a dungeon crawler against the clock.
Worked as game designer from the very beginning, helping with creating the idea, developing mechanics and building the different all the rooms. Also responsible for the technical parts of the design, such as giving indications on how the procedural layout generation should work and how the enemy ghosts AI should behave.
